Getting Started

Setup Typography

  1. Download Typography and add files to your website.
  2. Add css in the head of your webpage. 
    <link rel="stylesheet" href="css/typography.css">
<script src="http://code.jquery.com/jquery-1.10.1.min.js"></script>
<script type="text/javascript" src="js/typography.js"></script>
  3. Create typography in your html.

Customize Typography

  1. Fork or Download the Typography project on github.
  2. Make sure you have Sass and Compass installed.
  3. Edit the _options.scss with your own custom values
  4. In the root of the Typography directory type compass watch on the command line.
  5. The typography.css file should now be updated
